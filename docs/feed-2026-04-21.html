Summary

A curated set of 12 posts, papers, and projects selected for technical depth and relevance.

Papers

A Real-Time Bike-Pedestrian Safety System with Wide-Angle Perception and Evaluation Testbed for Urban Intersections

arXiv Vision | Mehmet Kerem Turkcan | 2026-04-18

This is one of the strongest items in the bundle. It presents a real-time urban safety system built around a wide-angle fisheye camera, and the paper goes well beyond detection by detailing a calibration pipeline, a lookup-table-based ground-plane projection stage, a conformance simulation framework, and explicit deployment-risk analysis. The combination of geometry, edge deployment, and testbench thinking makes it far more useful than a generic perception paper.

What's unique: It couples fisheye calibration, closed-form projection, and deployment gating in a single end-to-end safety system.

Fringe Projection Based Vision Pipeline for Autonomous Hard Drive Disassembly

arXiv Vision | Badrinath Balasubramaniam | 2026-04-19

This paper is a strong engineering piece because it uses structured-light style sensing through fringe projection and then integrates it with segmentation and depth completion for robotic disassembly. The pixel-wise alignment argument is especially compelling: the same camera-projector system supports both depth and semantic localization, which simplifies downstream 3D reasoning in a way many RGB-D pipelines do not.

What's unique: It combines fringe projection, depth completion, and segmentation in one inherently aligned 3D pipeline.

SemLT3D: Semantic-Guided Expert Distillation for Camera-only Long-Tailed 3D Object Detection

arXiv Vision | Hao Vo | 2026-04-20

SemLT3D is a solid camera-only 3D detection paper with a concrete angle on long-tail classes, which matters in safety-critical driving scenarios where rare categories can still be important. The semantic-guided expert routing and CLIP-informed distillation make it more interesting than a generic detector-improvement paper.

What's unique: It attacks long-tail camera-only 3D detection using semantic expert routing and semantic projection distillation.

Relative State Estimation using Event-Based Propeller Sensing

arXiv Robotics and Geometry | Ravi Kumar Thakur | 2026-04-20

This is a compelling niche paper because it uses event-based sensing of propellers for relative state estimation in UAV settings, which is an unusual sensing and geometry problem. It is exactly the kind of non-generic perception work that makes a digest feel curated rather than algorithmically assembled.

What's unique: It derives relative state from event-based observations of propeller motion, which is a very unusual sensing setup.

Projects

eowjd0512/fisheye-calib-adapter

GitHub Vision Geometry C++ | eowjd0512 | 2026-04-07

This repository is unusually relevant if you care about real camera models rather than generic OpenCV calibration scripts. It focuses on converting and reconciling multiple fisheye models, including double-sphere, Kannala-Brandt, EUCM, UCM, Kalibr, and WoodScape-style formulations, which is exactly the kind of glue code that shows up in real multi-camera and automotive perception stacks.

What's unique: It is about cross-model fisheye calibration interoperability, not just running one calibration routine.

fundamentalvision/BEVFormer

GitHub Automotive Perception | fundamentalvision | 2026-04-21

BEVFormer is already an important reference point for camera-only autonomous-driving perception, and it remains highly relevant if you are interested in BEV representations and multi-camera reasoning. Even though it is not new, it is one of the few GitHub items in the bundle that clearly belongs in an advanced automotive-vision digest.

What's unique: It is a canonical camera-only BEV perception framework rather than a small demo repository.

sourishg/fisheye-stereo-calibration

GitHub Vision Geometry C++ | sourishg | 2026-03-06

Stereo fisheye calibration is harder and more practically relevant than monocular calibration for many robotics and surround-view systems, so this repository earns a place on depth alone. It gives you a concrete C++ and OpenCV reference point for handling stereo geometry under fisheye distortion, which is exactly where many toy calibration guides stop being useful.

What's unique: It tackles stereo fisheye calibration directly, which is much closer to real multi-camera systems.

astar-ai/calicammono

GitHub Vision Geometry C++ | astar-ai | 2026-03-16

CaliCamMono looks like a practical tool for rectifying fisheye imagery into multiple output parameterizations such as cylindrical, perspective, and longitude-latitude projections. That makes it useful as a working reference for the downstream side of calibration, where the question is not only estimating parameters but also choosing how to remap the imagery for later perception tasks.

What's unique: It emphasizes multiple fisheye rectification targets rather than only calibration parameter estimation.

Posts

Oct 24, 2025 Narrow FOV Calibration Made Easy(er)

Tangram Vision Blog | 2026-04-21

This is one of the clearest matches for your interests in the entire bundle. Narrow-FOV calibration is a real engineering problem that regularly breaks downstream extrinsics and stereo systems even when reprojection error looks acceptable, and Tangram is writing directly about why that happens and how to handle it. That is exactly the right combination of theory and applied calibration pain.

What's unique: It focuses on the failure modes of narrow-FOV calibration rather than generic calibration basics.

Dec 9, 2025 Plane Fitting, Part 2: Manifold Optimization, without Lie Groups

Tangram Vision Blog | 2026-04-21

This is one of the best blog picks because it is directly about geometry and optimization rather than product promotion. Plane fitting is a deceptively deep topic in perception pipelines, and a post about manifold optimization without leaning on Lie-group boilerplate sounds exactly like the kind of mathematically grounded engineering note worth keeping.

What's unique: It revisits plane fitting through manifold optimization instead of the usual cookbook treatment.

GSoC 2025: Rich Disassembler for LLDB

LLVM Project Blog | 2025-11-10

This LLVM post is not directly about vision, but it clears the bar on technical substance. The project uses DWARF location ranges to enrich disassembly with variable-lifetime annotations, which is genuinely useful systems work if you care about low-level tooling, optimized binaries, and understanding what the debugger can recover from compiled code.

What's unique: It makes disassembly semantically richer by mapping machine instructions back to variable state.

How to Build Vision AI Pipelines Using NVIDIA DeepStream Coding Agents

NVIDIA Computer Vision Blog | 2026-04-21

DeepStream coding agents is a slightly odd article, but it still lands because it sits at the intersection of vision pipelines, deployment tooling, and NVIDIA’s production video stack. If your interest includes how vision systems get assembled and operated rather than only models, it is worth keeping.

What's unique: It treats vision-pipeline construction itself as an agent-assisted engineering workflow.
